# Prompts

Prompts que Martin usa y reutiliza.

## Cómo agregar uno
Dile a Claude: "guarda este prompt" y él preguntará nombre y categoría antes de guardarlo.

## Estructura sugerida
- Un archivo `.md` por prompt o por categoría
- Nombre descriptivo: `copywriting-anuncio-facebook.md`, `email-seguimiento-lead.md`
