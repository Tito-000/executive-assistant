## Introducción

WordPress.com es una plataforma de creación de sitios web basada en la nube que permite a usuarios sin experiencia técnica crear sitios web profesionales en cuestión de minutos. A diferencia de WordPress.org (que requiere hosting propio), WordPress.com ofrece una solución completamente alojada y administrada.

### ¿Por qué WordPress.com?

- **Accesibilidad:** No requiere conocimientos técnicos
- **Costo:** Plan gratuito disponible con subdominio .wordpress.com
- **Mantenimiento:** La plataforma se encarga de actualizaciones y seguridad
- **Facilidad:** Editor visual intuitivo mediante bloques
- **Escalabilidad:** Posibilidad de upgrade a planes pagos según necesidades

En este documento se documenta el proceso completo de creación de un sitio web simple desde cero.

## Paso 1: Registro e Inicio de Sesión

### 1.1 Acceso a WordPress.com

El primer paso es visitar **wordpress.com** en tu navegador web. La página de inicio muestra opciones para crear un nuevo sitio o iniciar sesión si ya tienes una cuenta.

**URL:** www.wordpress.com

### 1.2 Crear una Cuenta

Si no tienes cuenta, haces clic en **"Get started"** o **"Log in"** en la esquina superior derecha. WordPress.com ofrece varias opciones para registrarse:

- Correo electrónico y contraseña
- Iniciar sesión con Google
- Iniciar sesión con Apple
- Iniciar sesión con GitHub
- Enviar un enlace de inicio de sesión por correo

Para este ejemplo, se utilizó la opción de inicio de sesión con Google.

### 1.3 Autorización y Permisos

Después de autenticarse, WordPress.com requiere acceso a tu cuenta de Google. Esto es seguro y solo se usa para verificación de identidad.

**Pantalla de inicio de sesión:**
- La plataforma muestra una interfaz limpia con opciones de autenticación
- Se acepta la política de privacidad y términos de servicio
- Se obtiene acceso inmediato al dashboard

## Paso 2: Creación del Sitio

### 2.1 Iniciar Creación de Sitio

Una vez autenticado, se accede al dashboard principal. En la sección de "Sitios", aparece la opción **"Crear un sitio"** o "Añadir un nuevo sitio".

Haces clic en **"Crear un sitio"** para comenzar el proceso.

### 2.2 Seleccionar Opciones de Dominio

WordPress.com te presenta varias opciones:

1. **Comprar un dominio personalizado** (de pago)
2. **Usar un subdominio gratuito de WordPress.com**
3. **Usar un dominio que ya tienes registrado**

Para este ejemplo, se seleccionó la opción de **subdominio gratuito**. Esto crea un sitio con la URL:

```
nombre-del-sitio.wordpress.com
```

En este caso: `martinmercedes100-vhpce.wordpress.com`

### 2.3 Confirmación de Subdominio

Se ingresa el nombre deseado para el subdominio. WordPress.com valida disponibilidad automáticamente. Si el nombre no está disponible, sugiere alternativas.

**Nota:** El subdominio elegido fue `martinmercedes100-vhpce`, que combina el nombre solicitado con un identificador único para evitar conflictos.

## Paso 3: Selección de Tema

### 3.1 Exploración de Temas

Después de crear el sitio, WordPress.com presenta una galería de temas profesionales. Los temas son plantillas de diseño preconfiguradas que determinan la apariencia visual del sitio.

**Características de los temas disponibles:**
- Diseños modernos y responsivos
- Compatibles con dispositivos móviles
- Personalizables sin código
- Optimizados para SEO
- Múltiples categorías (negocios, portfolios, blogs, tiendas, etc.)

### 3.2 Selección del Tema

Para este proyecto, se seleccionó un tema limpio y minimalista con enfoque profesional. El tema incluye:

- Encabezado limpio
- Navegación intuitiva
- Sección de héroe (hero section) destacada
- Área para contenido principal
- Pie de página personalizable

### 3.3 Aplicación del Tema

Una vez seleccionado, WordPress.com aplica automáticamente el tema al sitio. El usuario es redirigido al editor visual para comenzar la personalización.

## Paso 4: Personalización de la Página Principal

### 4.1 Acceso al Editor Visual

WordPress.com utiliza un editor de bloques intuitivo. Cada elemento de la página (texto, imágenes, botones) es un "bloque" que se puede editar, mover o eliminar.

**Elementos accesibles en el editor:**
- **Bloques de texto:** Párrafos, títulos, citas
- **Bloques de media:** Imágenes, galerías, videos
- **Bloques interactivos:** Botones, formularios, embeds
- **Bloques de diseño:** Columnas, grupos, separadores

### 4.2 Edición del Título Principal

El sitio se creó con el título **"Martin Mercedes"** como encabezado principal de la página de inicio.

**Pasos seguidos:**
1. Hacer clic en el título existente
2. Editar el texto directamente
3. Personalizar tamaño de fuente
4. Ajustar alineación (centrado, izquierda, derecha)

### 4.3 Adición de Bloques de Contenido

Se agregaron varios bloques para crear contenido:

1. **Texto de bienvenida:** "Hello world!" como párrafo introductorio
2. **Bloque de presentación:** Sección dedicada para descripción del sitio
3. **Imagen destacada:** Se seleccionó una imagen de calidad profesional que muestra un moderno escritorio de diseño
4. **Texto descriptivo:** Se agregó descripción debajo de la imagen: "A sleek website for a design agency is showcased on a modern computer monitor in a bright office."
5. **Botón de acción:** Se agregó botón "Empieza ahora" con link de acción

### 4.4 Uso de IA para Generación de Contenido

WordPress.com ofrece herramientas de IA integradas. Se utilizó la función **"Mejorar con IA"** para:
- Sugerir descripciones automáticas
- Generar texto basado en prompts
- Optimizar contenido existente

**Captura de pantalla:** Se observa el editor mostrando la opción "¡Vamos a crear algo!" con un panel de generación de imágenes por IA que permite describir qué imagen se desea crear.

### 4.5 Organización del Contenido

El contenido se organizó en secciones lógicas:
- Encabezado con navegación
- Hero section con imagen
- Descripción de la agencia
- Call-to-action (botón de acción)
- Área de contenido adicional

## Paso 5: Creación de Páginas Adicionales

### 5.1 Estructura de Navegación Inicial

Se creó una estructura de navegación básica con dos páginas principales:

1. **Home (Página de Inicio):** La página principal del sitio
2. **Seleccionar una página:** Opción para agregar páginas adicionales

### 5.2 Adición de Página "Sobre Nosotros"

Se agregó una nueva página a la navegación:

**Pasos:**
1. Acceder al panel de navegación (izquierda)
2. Buscar opción "Agregar página"
3. Crear página con título "Sobre nosotros"
4. Agregar contenido específico
5. Publicar la página

### 5.3 Bloque de Navegación

WordPress.com mantiene un bloque de navegación que se actualiza automáticamente cuando se agregan nuevas páginas. Este bloque aparece en la parte superior de cada página del sitio.

**Navegación visible:**
```
Home | Sobre nosotros | [Seleccionar una página]
```

## Paso 6: Estructura de Navegación

### 6.1 Panel de Navegación

El panel izquierdo del editor muestra todas las páginas disponibles:

1. **Navegación:** Permite gestionar menús
2. **Lista de páginas:** Muestra todas las páginas creadas
3. **Mapa:** Visualización jerárquica del sitio

### 6.2 Gestión del Menú Principal

El menú principal se gestiona desde el panel de navegación. Se pueden:
- Agregar páginas al menú
- Reordenar páginas mediante arrastrar y soltar
- Crear submenús (páginas anidadas)
- Ocultar páginas del menú

### 6.3 Estado de las Páginas

Cada página tiene un estado:
- **Publicada:** Visible para visitantes
- **Próximamente:** En construcción
- **Privada:** Solo visible para dueño

## Paso 7: Publicación del Sitio

### 7.1 Revisión Previa

Antes de publicar, WordPress.com ofrece:
- **Vista previa en escritorio:** Cómo se ve en computadora
- **Vista previa móvil:** Cómo se ve en teléfono

### 7.2 Botón "Publicar"

Una vez satisfied con el contenido, se hace clic en **"Publica tu web"** (botón azul en la esquina superior derecha).

**Confirmación de publicación:**
- El sitio es accesible en la URL especificada
- Los cambios son visibles inmediatamente para visitantes
- Se puede seguir editando en cualquier momento

### 7.3 Panel de Control Post-Publicación

Después de publicar, WordPress.com muestra:
- URL del sitio publicado
- Opción para cambiar tema
- Acceso a análisis (si está disponible)
- Opciones de plan y upgrade

## Conclusiones y Reflexiones

### 9.1 Ventajas de WordPress.com Confirmadas

**A través de esta experiencia práctica, se confirman las siguientes ventajas:**

1. **Facilidad de uso:** El proceso de creación es intuitivo y requiere mínimo conocimiento técnico. Cualquier persona puede crear un sitio funcional en 30 minutos.

2. **Velocidad:** Desde registro hasta publicación, todo el proceso se completó en menos de una hora.

3. **Diseño profesional:** Los temas incluidos son modernos y se ven bien en todos los dispositivos.

4. **Escalabilidad:** Es fácil agregar más páginas, contenido y funcionalidades a medida que crezcan las necesidades.

5. **Soporte integrado:** WordPress.com maneja hosting, seguridad y backups automáticamente.

6. **Integración de IA:** Las herramientas de IA para generación de imágenes y contenido acelaran el proceso creativo.

### 9.2 Limitaciones Encontradas

1. **Dominios gratuitos:** El subdominio `.wordpress.com` no es ideal para marcas profesionales. Se recomienda comprar dominio personalizado ($17-20 USD/año).

2. **Funcionalidades limitadas en plan gratuito:** Características avanzadas requieren upgrade a plan de pago (Basic, Plus, Premium, Business).

3. **Moneterización limitada:** El plan gratuito no permite vender productos ni servicios fácilmente. Requiere plan Business.

4. **Personalización de código:** No se puede acceder a código HTML/CSS en plan gratuito. Requiere plan Business.

5. **Plugins limitados:** El plan gratuito restringe acceso a plugins. Requiere upgrade.

### 9.3 Casos de Uso Ideales para WordPress.com

**WordPress.com es ideal para:**

- **Blogs personales:** Escritores, fotógrafos, creadores de contenido
- **Portfolios:** Diseñadores, ilustradores, creativos
- **Sitios informativos:** Pequeños negocios, restaurantes, servicios locales
- **Sitios corporativos simples:** Empresas que necesitan presencia web básica
- **Tiendas pequeñas:** Con plan Business, se puede vender

**No es ideal para:**

- **Aplicaciones web complejas:** Requiere programación custom
- **E-commerce a gran escala:** Mejor usar Shopify o WooCommerce
- **Sitios con requisitos técnicos especiales:** Mejor usar WordPress.org

### 9.4 Próximos Pasos Recomendados

Para mejorar el sitio creado, se sugieren:

1. **Comprar dominio personalizado:** `martinmercedes.com` en lugar de subdominio
2. **Agregar más páginas:** Servicios, Portafolio, Contacto, Blog
3. **Contenido más detallado:** Descripciones completas de servicios
4. **Formulario de contacto:** Para capturar leads
5. **Integración con redes sociales:** LinkedIn, Instagram, etc.
6. **Blog:** Para SEO y autoridad
7. **Upgrade a plan Plus o Premium:** Para más funcionalidades
8. **Optimización SEO:** Títulos, descripciones, palabras clave
9. **Análisis de tráfico:** Usar estadísticas para mejorar contenido

## Apéndice: Glosario de Términos

- **Bloque:** Unidad de contenido en WordPress (texto, imagen, botón, etc.)
- **Tema:** Plantilla visual que define el diseño general del sitio
- **Plugin:** Extensión de funcionalidad (no disponible en plan gratuito)
- **Subdominio:** Parte del dominio antes del .wordpress.com
- **Dominio personalizado:** Dominio propio (ejemplo: ejemplo.com)
- **SEO:** Search Engine Optimization - optimización para buscadores
- **Responsive:** Sitio que se adapta a diferentes tamaños de pantalla
- **Hero section:** Sección principal destacada con imagen grande y texto
- **CTA:** Call-to-Action - botón o enlace que invita a actuar
- **Dashboard:** Panel principal de control del sitio
