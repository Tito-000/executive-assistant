# Onboarding GHL — Biblioteca de Videos para Clientes

> Programa de videos de entrenamiento para clientes de la agencia que usan GoHighLevel.
> El cliente recibe acceso a estos videos al hacer onboarding — así no tienes que explicar lo mismo dos veces.

**Origen:** Boris pidió cómo conectar Google Calendar → se grabó el video → se decidió hacer la serie completa.

---

## Cómo funciona

1. Cliente nuevo entra → recibe link a esta biblioteca
2. Ve los videos en orden (o busca lo que necesita)
3. Si tiene dudas después de ver el video → ahí sí te puede preguntar

## Dónde alojar los videos

Opciones (decidir antes de grabar):
- **YouTube privado/no listado** → links individuales por video
- **Loom** → más fácil de grabar, link directo
- **Google Drive** → simple, controlado
- **Página en GHL** → más profesional, dentro del mismo sistema

**Recomendación:** Loom para grabar rápido → después migrar a una página dentro de GHL si quieres que se vea más pro.
