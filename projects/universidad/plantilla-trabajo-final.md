## MÓDULO 6: MANEJADORES DE CONTENIDO PARA SITIOS WEB (CMS)

**(Mínimo 5 páginas)**

### 6.1 Marco Conceptual (¿Qué es un CMS?)

[ESCRIBE AQUÍ: Define con tus palabras...]

Puntos a cubrir:
- Definición de CMS (Content Management System)
- Tipos de CMS:
  - **Tradicional/Monolítico:** frontend y backend juntos (WordPress, Joomla, Drupal)
  - **Headless:** solo backend, entrega por API (Strapi, Contentful)
  - **Híbrido:** combina ambos (WordPress + API, Webflow)
  - **SaaS/Website Builders:** todo incluido (Shopify, Wix, Squarespace)
- Por qué un CMS es importante para una cadena farmacéutica

### 6.4 Recomendación

[ESCRIBE AQUÍ: Justifica tu elección...]

Recomendación sugerida: **WordPress + WooCommerce**

Razones para justificar:
1. CMS más usado del mundo (43% de sitios web — fuente: W3Techs)
2. Fácil encontrar desarrolladores en RD
3. WooCommerce maneja catálogo, inventario, pagos
4. Yoast SEO para posicionamiento en Google
5. Bajo costo vs Shopify o HubSpot
6. Plugins útiles: WPML (multi-idioma), WP Store Locator (sucursales), WhatsApp Chat
